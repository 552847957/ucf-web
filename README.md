# ucf-web 一个专注于性能与效率的前端微应用开发框架

> **ucf-web 是整个框架的代号，相关资源都在本仓库内维护，形成统一框架体系。**

## 关键特性

- 中台前端技术收敛化、标准化
- 三种微应用开发模式，打破铁桶一块的巨石应用
- UI 体验一致性，基于统一设计语言之上构建标准组件体系
- HTTP 通讯协议规范化，统一异常处理
- 可选择的模型驱动开发模式，基于元数据之上的统一快速开发

## 快速上手使用

```
$ npm i ucf-cli -g
$ ucf-cli init
$ cd xx && npm install
$ npm run dev
```
